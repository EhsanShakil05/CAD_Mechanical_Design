# Spur Gear and Gear Train Design

A CAD project developed in Fusion 360 to model spur gears, analyze gear ratios, and create a functioning gear train assembly.

## Features

- Parametric gear generation
- Gear ratio analysis
- Motion-linked assembly
- Engineering drawings
- Mechanical power transmission study

## Gear Parameters

- Driver Gear: 20 Teeth
- Driven Gear: 40 Teeth
- Module: 2 mm
- Pressure Angle: 20°
- Gear Ratio: 2:1

## Applications

- Robotics
- Gearboxes
- Power Transmission
- CNC Machines
- Mechanical Systems

## Software

- Autodesk Fusion 360
