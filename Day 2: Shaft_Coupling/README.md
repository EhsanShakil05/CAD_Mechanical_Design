# Shaft Coupling Design

A rigid shaft coupling designed in Fusion 360 to develop CAD modeling and mechanical component design skills.

## Features

- Parametric modeling
- Circular hole pattern
- Shaft bore design
- Fastener integration
- Engineering drawing generation

## Applications

- Power transmission
- Motors and gearboxes
- Mechanical systems
- Robotics assemblies

## Skills Demonstrated

- CAD Modeling
- Mechanical Design
- Engineering Drawings
- Component Design

## Software

- Autodesk Fusion 360
