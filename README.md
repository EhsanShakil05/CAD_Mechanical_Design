# CAD & Mechanical Design Portfolio

## Overview

A collection of mechanical design projects developed using Autodesk Fusion 360 as part of a Mechanical Engineering skills development program.

The portfolio focuses on component design, power transmission systems, robotics-related mechanical structures, engineering drawings, and design documentation.

---

## Projects Included

### Bearing Mount Design

* Parametric sketching
* Hole features
* Engineering drawings
* Bearing support design

### Shaft Coupling Design

* Power transmission component
* Circular patterns
* Mechanical fastening features
* Torque transmission design

### Spur Gear & Gear Train Design

* Gear generation
* Gear ratio analysis
* Motion relationships
* Mechanical assemblies

### Robot Chassis Design

* Robotics-focused mechanical design
* Component packaging
* Design for assembly
* Mobile robot platform development
