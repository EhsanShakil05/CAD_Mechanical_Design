# Robot Chassis Design

A differential-drive robot chassis designed in Fusion 360 for robotics and mechatronics applications.

## Features

- Arduino UNO mounting
- Motor mounting provisions
- Battery holder placement
- Weight reduction cutouts
- Assembly-focused design

## Applications

- Line Following Robots
- Obstacle Avoidance Robots
- Self-Balancing Robots
- Mobile Robotics

## Skills Demonstrated

- CAD Modeling
- Mechanical Design
- Design for Assembly
- Robotics Integration
- Engineering Drawings

## Software

- Autodesk Fusion 360
