# Bearing Mount Design

A simple mechanical component designed in Fusion 360 to develop CAD modeling and engineering drawing skills.

## Features

- Parametric sketching
- Fully constrained geometry
- Extrusion operations
- Hole features
- Engineering drawing generation

## Skills Demonstrated

- CAD Modeling
- Mechanical Design
- Engineering Drawings
- Dimensioning

## Software

- Autodesk Fusion 360
